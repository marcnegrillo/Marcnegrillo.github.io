Contact Form → Google Sheets

This workflow collects contact form submissions and automatically stores them in Google Sheets using n8n. It ensures all messages are saved with proper formatting and timestamps, making it easy to track and manage inquiries without manual work.

Workflow Examples

Workflow Steps

1. Form Trigger (On Form Submission)

A public contact form collects the following information:

  • Name
  • Email
  • Message

Once the form is submitted, the workflow is instantly triggered in n8n.

2. Form Completion Message

After submission, the user receives a confirmation message:

“Thank you! Your message has been submitted successfully.”

3. Edit Fields (Set Node)

The submitted data is cleaned and structured into the following fields:

  • Name
  • Email
  • Message
  • Date (automatically generated timestamp)

4. Google Sheets – Append Row

The processed data is appended as a new row in Google Sheets, creating a structured and organized record of all submissions.

Tools & Technologies

  • n8n
  • n8n Form Trigger
  • Google Sheets API
  • Webhook-based automation
