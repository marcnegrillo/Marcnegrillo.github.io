Contact Form → Google Sheets

This workflow collects contact form submissions and automatically stores them in Google Sheets using n8n. It ensures all messages are saved with proper formatting and timestamps, making it easy to track and manage inquiries without manual work.

Workflow Examples

Workflow Steps Form Trigger (On Form Submission) A public contact form collects: Name Email Message When the form is submitted, n8n is triggered instantly. Form Completion Message After submission, the user sees a confirmation message: “Thank you! Your message has been submitted successfully.” Edit Fields (Set Node) The submitted data is cleaned and structured: Name Email Message Date (automatically generated using the current timestamp) Google Sheets – Append Row The processed data is appended as a new row in Google Sheets, creating a structured log of all submissions.
